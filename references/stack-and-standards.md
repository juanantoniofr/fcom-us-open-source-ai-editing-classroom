# Stack y Estándares

## Stack base

- Linux.
- Kubernetes.
- vLLM, Ollama o TGI.
- PostgreSQL.
- Redis.
- MLflow.
- MinIO.
- OpenSearch.
- Keycloak.
- Prometheus, Grafana, Loki y Alertmanager.

## Estándares

- RGPD.
- LOPDGDD.
- ENS, si aplica.
- Políticas internas de la Universidad de Sevilla.

## Criterio de selección

Se priorizan componentes open source, sostenibles y con soporte suficiente para operación institucional.
